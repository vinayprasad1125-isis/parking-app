# 🚗 Parking Map App

A smart parking finder built using Google Maps API.

## Features
- 📍 Detects your current location
- 🗺️ Shows nearby parking spots (within 10km)
- 🟢 Available / 🔴 Occupied indicators
- 💰 Dynamic pricing per hour
- 🧾 Booking panel with hour selection
- 📱 Clean interactive UI

## Tech Stack
- HTML, CSS, JavaScript
- Google Maps JavaScript API

## Future Improvements
- Real parking data (API integration)
- Payment system
- User authentication
- Booking history
